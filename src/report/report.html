Import settings

RIM export relies on configuration of a set of DHIS 2 indicators. This page is used for selecting the relevant indicators and importing them into the DHIS 2. Administrators must then configure the indicator formulas to map each indicator to the appropriate data elements.

Indicators will be added to an indicator group called RIM Export, and the indicators will have [RIM] as prefix.

A new user group will be imported with the current users as member, and the RIM indicators will be visible only to user in that group. Only users in this group can thus configure the RIM indicators, and export data in RIM format. Additional users can be added to this group in the DHIS 2 user management app.
